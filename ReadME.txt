Application Run guide

For XAMPP

1. Make sure to copy source code folder to C:\xampp\htdocs
2. Then start Apache and MySQL services through The XAMPP control panel
3. Import the .sql file (Destine.sql) that we have attached with the project to the phpMyAdmin
